# 🍎 Apple Frameworks App

A SwiftUI application that explores Apple's ecosystem of frameworks through a modern, interactive interface.

## Features

- Browse Apple Frameworks
- Framework Details Screen
- Safari Integration
- MVVM Architecture
- SwiftUI Components
- Responsive UI

## Tech Stack

- Swift
- SwiftUI
- MVVM
- SafariServices
